I got a new system to make a homelab, this will have my Proxmox OS to manage all containers as needed. The Hardware specs are:
Specs:
- AMD ryzen 5 5600gt processor with radeon graphics(100-100001488Box)
- MSI B550M pro-VDH WIFI motherboard
- Corsair Vengence LPX 16GB (1 ram stick)
- PSU cooler master 450 watt bronze
- deep cool ag400 cpu cooler
- antec atx mid tower case
- evm 256 gb sata ssd

I plan to add more RAM and GPU in future for media library like jellyfin or plex.
