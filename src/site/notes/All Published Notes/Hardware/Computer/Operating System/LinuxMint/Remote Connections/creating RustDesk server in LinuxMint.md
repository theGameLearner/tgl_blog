### Theory
RustDesk is a third party solution, this works great with all platforms.
download the `.deb` file from [github](https://github.com/rustdesk/rustdesk/releases)
install with `sudo apt install -fy ./rustdesk-<version>.deb` file
open the app, you get the current machine's code and password.
#### Connecting over LAN or shared wifi (Viewer)
In ChromeOs, installed the `.deb` file with linux environment. open the app and use.

### Summary
Very easy to set up.
no port forwarding was needed.
no account needed.
best for local WiFi Connection.
